Crear nuevo repositorio en Git Hub

Una vez creada tu cuenta de Git Hub y haber descaragado Git:
- Creamos un nuevo repositorio, le agregamos descripción, si es público o privado.
-Vamos a la carpeta donde están los archivos que queremos subir.
-Hacemos click derecho, dentro de la carpeta y GIT BASH HERE.
-Eso abrirá Git y allí vamos a seguir estos pasos.

  1. git init /ENTER
  2. git add . /ENTER.
  3. git commit -m "mensaje"/ENTER
  4. git branch -M elnombredelbrach/ENTER
  5. git remote add origin https://github.com/Cassfsd/nuevo-repo.git/ENTER
    6. Esta línea te la da el Git Hub, podés copiar y pegar tal cual. Git no te permitirá hacer ctrl paste, así que debrás hacerlo con el click derecho.
  6. git push -u origin Casandra/ENTER
Recuerda que el nombre de las ramas debe coincidir,
